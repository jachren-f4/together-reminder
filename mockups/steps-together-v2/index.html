Steps Together v2

Redesign Overview

Comprehensive improvements based on UX evaluation

1 Fixed Ring Progress

Before: Each ring showed progress toward 20K individual steps. If both partners walk 5K each (10K combined = 15 LP), rings are only 25% full despite earning LP.
After: Single combined ring that fills as both partners contribute. Clear visual threshold at 10K where LP starts. Each partner's contribution shown as a segment.

2 Clearer Reward Messaging

Before: "Tomorrow's Reward" and "Tomorrow: +15 LP" was confusing. Users thought it was predicting tomorrow's steps.
After: "Today's Progress" with clear explanation: "Claim tomorrow for +15 LP". Added countdown to claim time.

3 Tier Breakdown

Before: Tier system (10K/12K/14K.../20K) mentioned but never explained. Users don't understand LP scaling.
After: Visual tier ladder showing all thresholds with LP values. Current tier highlighted. Next milestone clearly shown.

4 Warmer Partner Color

Before
Cold gray
After
Warm teal

5 Additional Improvements

  • Pull-to-refresh: Added swipe-down gesture indicator
  • Sync status: Prominent syncing animation in header
  • Milestone celebrations: Confetti burst when crossing tier thresholds
  • 7-day history: Week view showing past achievements and streaks
  • Auto-claim focused: Removed manual claim, celebration-only overlay
  • "Together" feeling: Encouraging team messages, combined progress emphasis
