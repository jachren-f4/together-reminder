Recommendation

Option 3 (Text Link) is recommended because:

  • Takes minimal vertical space (~30px vs ~60px for a card)
  • Visually signals it's a fallback, not a primary choice
  • Doesn't compete with the 5 main options for attention
  • Works on all screen sizes without scrolling
  • Easy to implement - just add a text button below the cards

The text "or: My response varies too much to choose" clearly positions it as an alternative while keeping it accessible for users who genuinely need it.
