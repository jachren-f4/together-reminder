Selected: Option A with Animation

Option A was selected because the animated bar creates excitement and clearly shows the connection between earning LP and progressing toward destinations. The animation sequence:

  1. Card fades in with pulsing heart emoji
  2. "+30 LP" badge pops in with gold gradient
  3. Bar fill animates from 0% toward the destination
  4. Heart indicator slides along with the fill
  5. Subtle glow pulse continues on the filled bar

Dynamic data: The destination image and name come from MagnetCollection so changing the first magnet in the config automatically updates this screen.

Implementation Notes

  • When shown: After Welcome Quiz completion, before home screen (existing lp_intro_overlay.dart)
  • Trigger: First time user completes welcome quiz and earns LP
  • Dismissal: "Let's Go!" button navigates to home screen
  • One-time: Only shown once per user (flag in local storage)
  • Fallback: If no magnet image available, show emoji placeholder
