Quiz Results Screen Options

The current design shows a percentage score (e.g., "60% MATCH"), which implies 100% is the goal. But the real value is discovering alignment AND differences—neither is better. Here are 5 alternative approaches.

Option A
Counts Only
Replace percentage with simple counts: "3 aligned · 2 different". No percentage means no implicit score or judgment.
3 aligned · 2 different
Removes percentage entirely
Option B RECOMMENDED
Two-Sided Celebration
Celebrate both outcomes equally: "3 things you share" and "2 things to explore together". Differences become opportunities, not failures.

Option C
Discovery Framing
Reframe as discoveries: "You discovered 3 alignments and 2 new insights about each other." Emphasizes learning over scoring.

Option D
No Summary
