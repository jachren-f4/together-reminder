# Sound Assets

This directory contains sound effects for the TogetherRemind app.

## Required Sounds

Download royalty-free sounds from these sources:
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/)
- [Freesound.org](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

### Criteria
- **Duration:** < 500ms (short clips)
- **Format:** MP3
- **Size:** < 50KB each
- **Style:** Subtle, soft, elegant feel

## Directory Structure

```
sounds/
├── ui/
│   ├── tap_soft.mp3        # Primary button tap
│   ├── tap_light.mp3       # Secondary button tap
│   ├── toggle_on.mp3       # Switch on
│   └── toggle_off.mp3      # Switch off
├── celebration/
│   ├── confetti_burst.mp3  # Main celebration
│   ├── sparkle.mp3         # Light achievement
│   └── chime_success.mp3   # Quest complete
├── feedback/
│   ├── success.mp3         # Action completed
│   ├── error.mp3           # Action failed
│   └── warning.mp3         # Attention needed
└── games/
    ├── card_flip.mp3       # Memory flip
    ├── match_found.mp3     # Match celebration
    ├── word_found.mp3      # Word search/linked
    ├── letter_type.mp3     # Linked typing
    └── answer_select.mp3   # Quiz answer
```

## Search Terms on Pixabay

- `ui click soft` - for tap sounds
- `toggle switch` - for toggle sounds
- `success chime` - for celebration sounds
- `notification soft` - for feedback sounds
- `card flip` - for game sounds
- `pop soft` - for match sounds
