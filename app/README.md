# togetherremind

A couples companion app for staying connected through games, reminders, and shared experiences.

## Features

### Home Screen (NEW)
The redesigned home screen features a gamified progression system with vacation-themed arenas:

- **Arena Progression**: 5 tiers from Cozy Cabin to Castle Retreat, each with unique gradients
- **Love Points System**: Earn LP by completing activities together
- **Quest-Based Interface**: Main Quests (games) and Side Quests (activities) categorization
- **Quick Actions**: Poke and Remind buttons for instant partner interaction
- **Live Stats**: Current Love Points, streak counter, and quiz match percentage
